An interactive **data analytics dashboard** built using **Streamlit, Pandas, Altair, and Machine Learning** to explore restaurant trends, customer preferences, and pricing insights.

---

## 🚀 Project Overview

This project analyzes restaurant data (inspired by Zomato datasets) to provide:

* 📊 Data-driven insights into food trends
* 📍 Location-based restaurant analysis
* 💰 Pricing and rating correlations
* 🤖 Machine Learning-based rating prediction

It is designed as a **portfolio-ready data science project** showcasing end-to-end skills:

> Data Cleaning → Analysis → Visualization → ML Modeling → Deployment

---

## ✨ Key Features

### 🔍 Interactive Filters

* Location-wise filtering
* Cuisine selection
* Cost range selection
* Rating and votes filtering
* High-rated restaurant toggle

---

### 📊 KPI Dashboard

* 🍴 Total Restaurants
* ⭐ Average Rating
* 💰 Average Cost
* 🗳️ Total Votes

✔ Animated cards with modern UI design

---

### 📈 Visualizations

* 🍽️ Top Cuisines Analysis
* 📍 Top Locations Analysis
* 💰 Cost Distribution
* ⭐ Avg Rating by Cost Range
* 📈 Cost vs Rating (Scatter Plot)
* 📊 Votes vs Rating (Scatter Plot)

✔ Built using **Altair interactive charts**

---

### 🤖 Machine Learning Model

* Predicts restaurant rating based on:

  * Cost
  * Votes
  * Cuisine count

**Model Performance:**

* 📉 R² Score ≈ 0.52

## ▶️ How to Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Zomato-Food-Delivery-Analytics.git
cd Zomato-Food-Delivery-Analytics
```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the Streamlit app

```bash
streamlit run dashboard/app.py
```
